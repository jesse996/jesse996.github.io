修改源

1
2
3

sudo pacman-mirrors -i -c China -m rank //更新镜像排名
sudo pacman -Syy //更新数据源
sudo pacman -S archlinux-keyring//更新 archlinux 签名

设置 archlinuxcn 源。

修改 /etc/pacman.conf 最后增加

1
2
3
4

[archlinuxcn]
SigLevel = Optional TrustedOnly
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

更新系统

1

sudo pacman -Syy

安装archlinuxcn-keyring:

1
2

sudo pacman -S archlinuxcn-keyring
sudo pacman -Syy

修改 Home 下的目录为英文：

修改目录映射文件名；

1

gedit ~/.config/user-dirs.dirs

修改为以下内容：

1
2
3
4
5
6
7
8

XDG_DESKTOP_DIR="$HOME/Desktop"
XDG_DOWNLOAD_DIR="$HOME/Downloads"
XDG_TEMPLATES_DIR="$HOME/Templates"
XDG_PUBLICSHARE_DIR="$HOME/Public"
XDG_DOCUMENTS_DIR="$HOME/Documents"
XDG_MUSIC_DIR="$HOME/Music"
XDG_PICTURES_DIR="$HOME/Pictures"
XDG_VIDEOS_DIR="$HOME/Videos"

将 Home 目录下的中文目录名改为对应的中文名；

1

cd ~

mv 音乐 Music

重启系统。

##安装 chrome

1

sudo pacman -S google-chrome

安装和使用 oh-my-zsh

安装 oh-my-zsh 的配置

查看本地有哪几种 shell

1

cat /etc/shells

manjaro 默认已经安装了 zsh

安装 oh-my-zsh 的配置文件

1
2
3
4
5

#via curl
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

#或者 via wget
sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"

替换 zsh 的配置文件为 oh-my-zsh

1

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

安装插件

安装 zsh-autosuggestions

1

git clone https://github.com/zsh-users/zsh-autosuggestions ~/.oh-my-zsh/plugins/zsh-autosuggestions

安装 zsh-syntax-highlighting

1
2


git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ~/.oh-my-zsh/plugins/zsh-syntax-highlighting

安装 zsh-nvm

1

git clone https://github.com/lukechilds/zsh-nvm ~/.oh-my-zsh/custom/plugins/zsh-nvm

安装 autojump

1

sudo pacman -S autojump

在~/.zshrc中找到

1
2
3

plugins=(
  git
)

在括号中 git 的下一行添加插件名称使其生效

1
2
3
4
5
6
7

plugins=(
  git
  zsh-syntax-highlighting
  zsh-autosuggestions
  autojump
  zsh-nvm
)

更改主题&&lazyload nvm

1

gedit ~/.zshrc

1
2

ZSH_THEME="agnoster"
export NVM_LAZY_LOAD=true

刷新配置，使之生效

1

 source ~/.zshrc

油酸乳

下载 脚本文件
1

wget http://www.djangoz.com/ssr#//github.com/the0demiurge/CharlesScripts/blob/master/charles/bin/ssr

ssr  是一个 shell 脚本

执行脚本
1
2
3
4

 sudo mv ssr /usr/local/bin
 sudo chmod 766 /usr/local/bin/ssr
 ssr install
 ssr config

然后在系统设置里设置网络代理为手动

开机自启动

在/usr/lib/systemd/system/中加入ssr.service

1
2
3
4
5
6
7
8
9

[Unit]
Description=AutoRunSSR

[Service]
Type=forking
ExecStart=ssr start

[Install]
WantedBy=multi-user.target

然后输入命令

1
2

systemctl enable ssr.service
systemctl start ssr.service

可以查看状态

1

systemctl status ssr.service

设置 chrome 为默认浏览器:

在设置中把 html 关联类型设置为用 chrome 打开
